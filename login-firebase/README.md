# Login com Firebase

### Objetivo

Exemplo de Login E-mail/Password no React, utilizando o serviço do Firebase.

Link CodeSandbox: https://codesandbox.io/s/loginfirebase-src9d

### Tecnologias utilizadas:

 - Firebase
 - Material-UI
 - YUP (p/ validação dos campos)
 - React Router
